# HTML/CSS/JS Kod Kaynakları

Bu liste, kopyala-yapıştır için HTML, CSS ve JS kod parçaları, butonlar, animasyonlar, tam site şablonları ve UI bileşenleri sunan siteleri içerir. Siteler kategorilere ayrılmıştır.

---

## Kod Parçaları / Snippet Siteleri

1. [CodePen](https://codepen.io/) - HTML, CSS ve JS denemeleri için canlı ön yüz platformu. Kodları “Fork” ile kopyalayabilirsiniz.  
2. [CodeMyUI](https://codemyui.com/) - Buton, slider, loader ve animasyon gibi seçilmiş UI parçaları.  
3. [FreeFrontend](https://freefrontend.com/) - Slider, form, buton gibi kategorize edilmiş küçük bileşenler.  
4. [CSS-Tricks Snippets](https://css-tricks.com/snippets/) - Kullanışlı CSS ve HTML kod parçaları.  
5. [CSSDeck](https://cssdeck.com/) - CSS ağırlıklı küçük animasyon ve efekt örnekleri.  
6. [Animista](https://animista.net/) - CSS animasyonlarını görsel olarak oluşturup kodu dışarı alabilirsiniz.  
7. [Webleb](https://www.web-leb.com/) - HTML, CSS ve JS UI elemanları, copy-paste ile kullanılabilir.  
8. [FreeWebSnippets](https://freewebsnippets.com/) - Başlık, buton, kart gibi hazır kod parçaları.  
9. [CSSnippets](https://dev.to/devshefali/all-the-code-snippets-you-need-at-one-place-cssnippets-745) - HTML, CSS, React ve Tailwind bileşenleri.  
10. [CSS Loaders](https://css-loaders.com/) - Farklı tipte yükleme animasyonları.  
11. [Neumorphism.io](https://neumorphism.io/) - Modern neumorphism stilinde animasyonlu buton ve kartlar.  
12. [Getwaves.io](https://getwaves.io/) - SVG dalga efektleri için kopyala-yapıştır kodlar.  
13. [CSS Grid Generator](https://cssgrid-generator.netlify.app/) - CSS grid şablonları üretip kopyalayabilirsiniz.
